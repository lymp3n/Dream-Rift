# Тайминг боя

## Система времени хода

### Базовые параметры
- **Первый ход**: 15 секунд
- **Уменьшение**: 10% каждый ход
- **Минимум**: 5 секунд

### Формула
```
Время хода = max(5, 15 * (1 - 0.1 * (номер_хода - 1)))
```

### Примеры
- Ход 1: 15.0 секунд
- Ход 2: 13.5 секунд (15 * 0.9)
- Ход 3: 12.15 секунд (15 * 0.81)
- Ход 4: 10.935 секунд (15 * 0.729)
- ...
- Ход 11+: 5.0 секунд (минимум)

## Философия дизайна

### Почему уменьшение времени?
1. **Напряжение**: Бой становится интенсивнее
2. **Навык**: Требует быстрых решений
3. **Динамика**: Предотвращает затягивание боя

### Почему минимум 5 секунд?
- Достаточно для принятия решения
- Не слишком стрессово
- Позволяет использовать навыки

## Навыки и время хода

### Навыки, расходующие ход
- Обычная атака
- Мощный удар
- Огненная Вспышка
- Лечение

### Навыки, НЕ расходующие ход
- Быстрая атака (требует тактику атаки)
- Защитная стойка (требует тактику блока)
- Магический щит (требует только ману)

## Стратегия тайминга

1. **Первые ходы**: Используйте время для планирования
2. **Средние ходы**: Быстрые решения, накопление тактик
3. **Поздние ходы**: Используйте навыки без расхода хода для экономии времени
