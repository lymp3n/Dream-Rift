# Формулы и баланс

## Формулы характеристик

### Максимальное HP

```
Макс. HP = (30 + (Уровень * 10)) + (STR * 3) + (END * 5)
```

**Пример:**
- Уровень 5, STR 15, END 12
- HP = 30 + 50 + 45 + 60 = 185

### Максимальное MP

```
Макс. MP = (20 + (Уровень * 5)) + (INT * 4) + (WIS * 2)
```

**Пример:**
- Уровень 5, INT 18, WIS 12
- MP = 20 + 25 + 72 + 24 = 141

### Физический урон

```
Базовый урон = (Сила * 2) + Бонус оружия
Финальный урон = Базовый урон ±15%
```

**Пример:**
- STR 15, оружие +5
- Базовый = 15 * 2 + 5 = 35
- Диапазон: 30-40 (35 ± 15%)

### Магический урон

```
Базовый урон = (Интеллект * 1.8) + Бонус
Финальный урон = Базовый урон ±10%
```

**Пример:**
- INT 20, бонус +3
- Базовый = 20 * 1.8 + 3 = 39
- Диапазон: 35-43 (39 ± 10%)

### Физическая защита

```
Физ. Защита = (Выносливость * 1.5) + Бонус брони
```

Работает как **прямое вычитание** из входящего физ. урона.

**Пример:**
- END 12, броня +5
- Защита = 12 * 1.5 + 5 = 23
- Удар на 50 урона → получаемый урон = 50 - 23 = 27

### Магическая защита

```
Маг. Защита = (Мудрость * 0.8) + Бонус
```

Работает как **процентное снижение** входящего маг. урона.

**Пример:**
- WIS 15, бонус +3
- Защита = 15 * 0.8 + 3 = 15%
- Удар на 50 маг. урона → получаемый урон = 50 * (1 - 0.15) = 42.5

### Шанс критического удара

```
Шанс Крита = (Ловкость * 0.1) + (Удача * 0.05) + Бонусы
```

**Пример:**
- AGI 12, LUK 10, бонус +2
- Шанс = 12 * 0.1 + 10 * 0.05 + 2 = 1.2 + 0.5 + 2 = 3.7%

### Критический урон

```
Крит урон = обычный урон * 1.8
```

**Пример:**
- Обычный урон: 50
- Крит урон: 50 * 1.8 = 90

### Скорость

```
Скорость = (Ловкость * 0.7) + Бонусы
```

Определяет порядок хода в бою. Персонаж с большей скоростью ходит первым.

**Пример:**
- AGI 15, бонус +3
- Скорость = 15 * 0.7 + 3 = 10.5 + 3 = 13.5

## Система дропа

### Влияние LUK на дроп

```
Финальный шанс дропа = Базовый шанс + (LUK * 0.0001)
```

Каждая точка LUK увеличивает шанс дропа на 0.01%.

**Пример:**
- Базовый шанс Сердцевины: 0.001 (0.1%)
- LUK персонажа: 20
- Финальный шанс: 0.001 + (20 * 0.0001) = 0.003 (0.3%)

### Вероятности дропа

- **Сердцевина (Core)**: 0.01% - 0.1%
- **Оболочка (Shell)**: 0.5% - 1%
- **Обычные предметы**: 5% - 50%

## Бонусы классов

### Рыцарь Костей
- +2 STR за уровень
- +2 END за уровень

### Маг Бездны
- +3 INT за уровень
- +1 WIS за уровень

### Странник Снов
- +1 INT за уровень
- +1 WIS за уровень
- +1 LUK за уровень

### Безклассовый
- +1 ко всем статам за уровень
