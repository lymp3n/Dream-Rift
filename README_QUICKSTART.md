# Быстрый старт

## Установка и запуск одной командой

```bash
python run.py
```

Скрипт автоматически:
1. Проверит и установит зависимости
2. Инициализирует базу данных
3. Запустит бэкенд сервер
4. Запустит CLI интерфейс

## Ручная установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Инициализация БД

```bash
python -m backend.src.database.init_db
```

### 3. Запуск бэкенда

```bash
cd backend
uvicorn backend.src.api.main:app --reload
```

### 4. Запуск фронтенда

В другом терминале:

```bash
cd frontend
python -m frontend.cli.main_modern
```

## Новые возможности

### Улучшенная система боя
- Пошаговый бой с таймингом (15 секунд первый ход, уменьшается на 10% каждый ход)
- Выбор между обычной атакой и навыками
- Некоторые навыки не расходуют ход
- Система тактик (атака, блок, крит и др.)

### Система тактик
- **Тактика атаки**: Получается после успешной незаблокированной атаки
- **Тактика блока**: Получается после успешного блока
- **Тактика критической атаки**: Получается после крита
- И другие...

### Современный интерфейс
- Чат в нижней части экрана
- Свайп вверх для открытия чата на весь экран
- Улучшенная визуализация боя
- Прогресс-бары для HP/MP

### Команды чата
- `/chat` или `/c` - открыть/закрыть чат
- `/clear` - очистить чат

## Тестовые навыки

1. **Мощный удар** - требует тактику атаки + немного маны
2. **Быстрая атака** - не расходует ход, требует тактику атаки
3. **Защитная стойка** - не расходует ход, требует тактику блока

## Агент-планировщик задач

Используйте агента-планировщика для разбивки проекта на задачи:

```bash
python -m orchestrator.task_planner
```

Он создаст файл `orchestrator/tasks.json` с задачами и подзадачами.
